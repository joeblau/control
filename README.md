# Control

Control your iOS Simulator

![hero](.github/assets/control-hero.png)

## Build

Prerequisites

- [Brew](https://brew.sh)
- [Xcodegen](https://github.com/yonaskolb/XcodeGen)

```sh
git clone git@github.com:ridevelo/control.git
cd control
xcodegen
```
